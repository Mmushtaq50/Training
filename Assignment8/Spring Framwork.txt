What is Spring MVC?

	Spring MVC FrameWork provides Model-View-Controller architectur to develop flexible 
	and loosely coupled Web Applications.
	
	The MODEL- Application Data(POJO class).
	The VIEW - HTML page seen by the End User.
	the CONTROLLER - Responsible for processing the user Request and building model 
			and passes it to the view of renderring.

What is Dispatcher Servlet / Front Controller?
   
 	Dispatcher Servlet is a front controller that handles all the incomming HTTP request 
		and send back the response to client.

	   Req				------------->   CONTROLLER
	  --------->
CLIENT	              DISPATCHER_SERVLET                VIEW_RESOLVER
	 <----------							
	    Resp			 <-------------     VIEW

Spring IOC container and its Advantages?

	1.Spring IOC (Inversion of Control) is a core of a Spring Framework.	
	2.The Container will read the configurations files and based on that it will 
		create and store the Objects and manage it.

	Advantages: 
	
	1.Developers can concentrate on bussiness logic since Object creation and managing
		are handled by IOC container.
