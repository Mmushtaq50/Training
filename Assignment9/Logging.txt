Logging:
	Spring Boot uses Apache Commons Logging for all internal logging , but leaves the underlying log implementation open.
	Default configurations are provided for Java Util Logging, Log4J2, and Logback
	
	1.the basic way to use logging is loggerFactory and it has a method called getLogger
		Logger logger = LoggerFactory.getLogger(classname.class);
