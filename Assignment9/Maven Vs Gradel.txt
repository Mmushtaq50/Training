Maven:
        1.Maven is based on developing pure Java language-based software. 
	2.It uses Extensible Markup Language(XML) for creating project structure
	3.It is necessary to compile.
	4.This tool is a known tool for many users and is easily available.

Gradel: 
	1.Gradle is based on developing domain-specific language projects.
	2. It uses a Groovy-based Domain-specific language(DSL) for creating project structure
	3.It avoids compilation.
	4.It is a new tool, which requires users to spend a lot of time to get used to it.
